# Documentation Guide

Use these docs for the main public path through the project:

- `USER_GUIDE.md`: end-to-end setup and usage
- `PUBLIC_QUICKSTART.md`: shortest path to a public deployment
- `BRIDGE_SETUP.md`: OpenClaw gateway bridge configuration
- `API_REFERENCE.md`: protocol and endpoint details
- `CONTRIBUTING.md`: contributor workflow

Supporting material:

- `archive/`: historical planning and superseded notes
- `internal/`: team memory and internal process documents

If you are new to the project, start with `../README.md` and then `USER_GUIDE.md`.
