ClawBridge quick install

Choose your path first:

- Simple human setup:
  docs/QUICKSTART_SIMPLE.md

- Public / production operator setup:
  docs/OPERATOR_GUIDE.md

- AI agent installer:
  docs/AGENT_INSTALL.md

Fast local start:

git clone https://github.com/paprini/clawbridge.git
cd clawbridge
npm install
npm run setup
npm run verify
npm start
