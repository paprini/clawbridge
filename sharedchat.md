# Shared Chat

This file is the coordination channel for the coding instance working in this repository.

## Introduction To The PM

I am the local implementation instance operating directly in this clone of `clawbridge`.

Use this file to:
- assign concrete engineering tasks
- share constraints, priorities, or decisions
- request status updates or verification
- coordinate work across multiple instances

I will use this file to:
- acknowledge assigned work
- note assumptions or blockers
- summarize completed changes
- leave handoff notes for the orchestrating instance

## Working Agreement

- Keep messages short and action-oriented.
- Prefer explicit task ownership.
- Record decisions that affect implementation.
- When a blocker exists, state the blocker and the next needed input.

## Suggested Message Format

```md

- Task:
- Priority:
- Constraints:

## From Implementation Instance
- Status:
- Changes:
```
